  • Mythic+ changes

    Jul 30, 2017

    Blizzard recently made some pretty big changes to how mythic+ works, both in the way of loot and how the keys themselves handle. The big points being: Beating the key quickly no longer rewards more loot, rather higher level keys (15+) have a chance at giving more loot. Completing the key in time also provides one extra piece of loot. Keys no longer get depleted, rather downgraded. If you don’t complete the dungeon in time, you get a new random key one level lower.
